Current projects

  • Dissertation: Composite function networks
    • The general trend in pattern recognition seems to be that successful models involve transforming the input space, sometimes more than once. Some ensemble methods and neural networks could be framed like this: in terms of iterating a map(s) with adjustable parameters.
    • The main goal is to inquire about two properties of machine learning models: the class of functions that a particular model structure can compute, and the role of symmetries. Starting with one-dimensional functions and operators, this project carries on into image data, which brings into the picture methods applicable in computer vision.
    • This project requires maths from different areas such as optimization, functional analysis, and digital signal processing.
  • Metaheuristic construction of latent space using genetic programming.
    • The idea is to inquire about the possibility of detecting unsupervised patterns with a graph and nonlinear functions at the vertices. Genetic programming is used as a feature selection process. There are some cluster shapes that are very easily detected with a previous feature transformation, for example, coordinate transformations. There are additional benefits from using genetic programming as a metaheuristic, like its use of dimensionality for one-vs-all class separation, and high-order polynomial transformations by taking advantage of the combinatorial structure of tree search, analogous to the growth of terms in polynomials and series expansions.

Other projects

  • Investment portfolio risk analysis using generalized Pareto distribution (Spanish language, 2018).
    • A brief tutorial on the use of extreme value theory to estimate value at risk of a Pareto-optimal portfolio.
  • Time series forecasting using rules ensemble.
    • The aim of this project is to search for consistent pattern rules that are good predictors, based on N. Packard's idea of using conditions that are met historically. It is still unfinished, because some more analysis and a better architechture are still pending before drawing conclusions. This analysis is intended to inquire on what results in efficient statistics and pattern recognition; for example, data preprocessing, rule syntax, use of ensembles, distribution model, objective function, etc.
  • Implementation of adiabatic quantum algorithms using classical light states (2018).
    • Motivation for implementing quantum algorithms with classical light comes from the fact that the Schrödinger equation is exactly the same as the paraxial equation: they both describe wave phenomena. This fact has already been exploited to implement for example the Deutsch-Joza and Grover's algorithm.
    • The main focus of the project was to implement an algorithm outside the gate-model quantum computer. The most promising setup involved a resonator cavity, where the evolving state was composed of a grid of OAM vortices. The project explored the possibility of using the evolution of the OAM state to implement quantum algorithms.
